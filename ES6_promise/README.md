# ES6 Promises

This project covers ES6 Promises in JavaScript.

## Learning Objectives

- Promises (how, why, and what)
- How to use the `then`, `resolve`, `catch` methods
- How to use every method of the Promise object
- Throw / Try
- The await operator
- How to use an async function

## Requirements

- Ubuntu 20.04 LTS
- NodeJS 20.x.x
- npm 9.x.x

## Setup

```bash
npm install
```

## Usage

```bash
npm run dev 0-main.js
npm run test
npm run full-test
```
